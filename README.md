# 📊 Dataset Health Analyzer

A professional **Streamlit-based web application** that automatically audits dataset quality, calculates a **Data Health Score**, evaluates **Machine Learning readiness**, and provides **one-click dataset cleaning** with downloadable reports.

Designed for **Data Scientists, Data Analysts, Students, and ML Engineers**, this application helps identify data quality issues before model development or business analysis.

# ✨ Features

## 📂 Dataset Upload

- Upload CSV datasets
- Upload Excel (.xlsx) datasets
- Instant dataset preview

---

## 📊 Data Profiling

Automatically displays:

- Number of Rows
- Number of Columns
- Dataset Size
- Memory Usage
- Column Names
- Data Types

---

## 🔍 Data Quality Audit

Automatically detects:

- Missing Values
- Duplicate Records
- Outliers (IQR Method)
- Constant Columns
- High Cardinality Features
- Invalid Data Types

Data Scientists spend nearly **80% of their time cleaning and preparing datasets** before building Machine Learning models.

Manual data auditing is repetitive, time-consuming, and error-prone.

Dataset Health Analyzer automates this process by providing:

- Automated quality assessment
- Data Health Score
- Machine Learning readiness evaluation
- Cleaning recommendations
- One-click dataset preprocessing

This significantly reduces the time required for data preparation.
